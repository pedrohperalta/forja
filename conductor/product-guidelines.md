# Product Guidelines — Forja

## Voice and Tone

**Friendly and approachable.** The app communicates in a casual, encouraging way — like a gym buddy, not a personal trainer certification manual. All UI text is in Portuguese (pt-BR).

### Guidelines

- Use informal "você" form
- Keep messages short — the user is between sets
- Celebrate progress without being over the top
- Error messages should be helpful, not technical

## Design Principles

### Simplicity Over Features

- Every screen has one clear purpose
- Minimize taps to complete any action
- No settings screens, no configuration — it just works
- Large touch targets for sweaty hands and gym gloves
- Linear flow: the app guides the user, not the other way around

### In-Context Design

- Designed to be used **during** the workout, not before or after
- One-hand operation is a hard requirement
- Timer and rest periods are first-class citizens
- The app should never block the user's flow

## UI Language

- All user-facing text: **Portuguese (pt-BR)**
- Code, documentation, and Conductor files: **English**
