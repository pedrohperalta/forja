01 — Resumo

Resumo Executivo

Forja é um app mobile pessoal que resolve um problema comum de iniciantes na academia: a falta de registro estruturado dos treinos. Sem anotações, é impossível saber se houve sobrecarga progressiva — o principal motor do ganho de massa muscular.

O app segue um plano de hipertrofia ABC (3 sessões, 7 exercícios cada) prescrito pela academia Espaço Prime. A interface é 100% em português, funciona offline, e foi desenhada para ser usada durante o treino — com botões grandes, timer de descanso, e fluxo linear sem distrações.

02 — Problema

Declaração do Problema

Iniciantes na academia enfrentam barreiras que vão além do esforço físico. A falta de organização e feedback sobre o progresso leva à estagnação e desmotivação.

Sem registro de treinos

Pedro não anota pesos ou séries. A cada sessão, precisa tentar lembrar o que fez na semana anterior — geralmente sem sucesso. Resultado: usa a mesma carga por semanas, sem progressão.

Apps existentes são complexos demais

Apps como Strong, Hevy e JEFIT têm dezenas de funcionalidades voltadas para intermediários/avançados. Para um iniciante com um plano fixo de 3 treinos, a complexidade é uma barreira, não uma ajuda.

Caderno e bloco de notas são lentos

Anotar pesos no celular (app de notas) ou em papel é desorganizado, lento, e difícil de consultar depois. Não existe estrutura, timer, ou histórico consultável.

Aparelhos ocupados quebram o fluxo

Na academia, é comum um aparelho estar ocupado. Sem uma forma fácil de pular e voltar depois, o fluxo do treino se perde, exercícios são esquecidos.

04 — Fluxo

Fluxo do Usuário

O app segue um fluxo linear durante o treino. O usuário seleciona o treino, executa séries com descanso cronometrado, e recebe um resumo ao final.

Loop de Séries

Cada exercício tem 3 séries. Após cada série (incluindo a última), o app mostra a tela de descanso com timer de 60 segundos. Após o descanso da última série, avança para o próximo exercício.

Série 1 → Descanso → Série 2 → Descanso → Série 3 → Descanso → Próximo Exercício

Fluxo de Pular Exercício

Se o aparelho está ocupado, o usuário pode pular o exercício atual. O exercício vai para o final da fila. Um indicador vermelho mostra quantos exercícios foram pulados. Na tela de descanso, um botão permite voltar aos exercícios pulados.

5.3 Tela de Descanso

P0 Timer de descanso
Dado que o usuário completou uma série,
Quando a tela de descanso aparece,
Então um timer circular de 60 segundos inicia com contagem regressiva animada.
P1 Alerta nos últimos 10 segundos
Dado que o timer tem menos de 10 segundos,
Quando o timer atualiza,
Então a cor muda para vermelho (#FF453A) para sinalizar urgência.
P1 Vibração ao terminar
Dado que o timer chegou a zero,
Quando o descanso termina,
Então o celular vibra (padrão 200ms-100ms-200ms) e o app avança automaticamente.
P1 Pular descanso
Dado que o usuário quer continuar antes do timer,
Quando toca "Pular Descanso",
Então o timer é cancelado e o app volta para a próxima série ou exercício.

State Machine do Treino

A rota app/workout/[id].tsx gerencia o treino inteiro com um estado interno:

type Phase = 'exercise' | 'rest' | 'complete';

// Estado principal:
phase: Phase
exerciseQueue: number[]      // índices restantes (ordem mutável)
skippedIndices: number[]     // índices pulados
currentSetIndex: number      // 0..2 dentro do exercício atual
completedLog: ExerciseLog[]  // exercícios completados
startMs: number              // timestamp de início

10 — Métricas

Métricas de Sucesso

Como app pessoal de um único usuário, as métricas são qualitativas e focadas em hábito e utilidade real.

Uso Regular

O app é usado em todos os treinos da semana (3x) sem recorrer a alternativas como bloco de notas.

Sobrecarga Progressiva

É possível verificar se os pesos aumentaram ao longo das semanas consultando o "último peso" de cada exercício.

Velocidade de Uso

Registrar uma série (inserir peso + tocar botão) leva menos de 5 segundos durante o treino.

Treinos Completos

O fluxo de pular exercícios permite completar o treino inteiro mesmo quando aparelhos estão ocupados.
