# Discovery Conversation — Training Configuration

## Q1: What type of idea?
**A:** New feature

## Q2: Describe the feature
**A:** A training configuration feature. I wanna be able to create/edit/remove training days, exercises from the training, repetitions, images, rest time, and every other parameter I need when my training changes over time.

## Q3: Plan count — fixed 3 or flexible?
**A:** Flexible (any number) — users can add/remove training days freely (e.g., 2-day split, 5-day split, etc.)

## Q4: Exercise source — library or custom?
**A:** Fully custom — users type in whatever exercise name they want, total freedom.

## Q5: What kind of images?
**A:** Equipment photos — keep the existing camera feature for exercise equipment reference photos.

## Q6: Which parameters per exercise?
**A:** Name, sets, reps, rest time, category, and equipment type labels.

## Q7: Where does configuration live in navigation?
**A:** Dedicated config screen — a new top-level screen (e.g., "Meus Treinos") accessible from home, separate from the workout flow.

## Q8: Should edits affect workout history?
**A:** History is immutable — past workouts always show the plan as it was when completed. Edits only affect future workouts.

## Additional Notes
- No S3 sync — keep files local in the docs folder.
- Session ID without date suffix.
