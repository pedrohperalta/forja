# Smart Training Import

## Raw Idea
Adding trainings and exercises is very time consuming and error prone. Given that it changes over time, I'd like to easily input this information into the app.

## Discovery Context
- **Change driver**: Coach sends new training program periodically
- **Input format**: Coach delivers programs as PDF or image
- **Program size**: 2-3 plans (A/B/C), 5-7 exercises each
- **Update style**: Replace everything at once when new program arrives
- **Pain points**: Typing exercise names (typos, remembering exact names) and entering all details (sets, reps, rest, equipment) for every exercise
- **Preferred solution**: Combination approach — AI scan for initial setup + exercise catalog for tweaks and future edits
